== Description ==

The simplest way to make your video embeds responsive without any shortcodes or added markup. Simply install the plugin and then proceed to embed videos as normal.

== Installation ==

1. Upload simple-responsive-videos.zip to plugins via WordPress admin panel, or upload unzipped folder to your plugins folder
2. Activate the plugin through the “Plugins” menu in WordPress
3. Toast your success!
